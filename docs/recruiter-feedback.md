# TraceMind AI Recruiter-Specific Survey

This version is designed for non-technical talent review. It focuses on first impression, professionalism, clarity, and whether the project supports a strong candidate profile.

## Suggested Questions

### 1. Based on a quick review, how clear is the purpose of TraceMind AI?

Response type: `1-5 rating`

### 2. How professional does the GitHub project look overall?

Response type: `1-5 rating`

### 3. How easy was it to understand what the candidate built?

Response type: `1-5 rating`

### 4. Does the README help you understand the project quickly?

Response type: `1-5 rating`

### 5. Does this project make the candidate look organized and detail-oriented?

Response type: `1-5 rating`

### 6. Does this project suggest the candidate can present technical work clearly to others?

Response type: `1-5 rating`

### 7. What part of the project gave the strongest impression?

Response type: `short answer`

### 8. What part was confusing or took too long to understand?

Response type: `short answer`

### 9. Would this project make you more likely to move the candidate forward to a screening conversation?

Choose one:

- Yes
- Maybe
- No

### 10. What would make this portfolio project more compelling from a hiring perspective?

Response type: `paragraph`

## What This Survey Helps Measure

- First-impression clarity
- Candidate professionalism
- Communication quality
- Portfolio credibility
- Screening interest
